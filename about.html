About StudentTasker

A simple, effective tool to help students stay organized and productive.

🎯

Our Purpose

StudentTasker was created to solve a common problem faced by university students — keeping track of multiple assignments, study schedules, and personal tasks. Instead of juggling sticky notes, spreadsheets, and reminder apps, students can now manage everything in one clean, distraction-free dashboard.

🎓

Who Is It For?

This app is designed specifically for university and college students who want a lightweight, no-fuss task manager. Whether you're tracking homework deadlines, exam preparation sessions, or personal errands — StudentTasker keeps it all organized with categories like Study, Assignment, and Personal.

💻

Technology Stack

This project is built using only frontend web technologies:

🟧 HTML5
🟦 CSS3
🟨 JavaScript (ES6)
💾 localStorage API
✨

Key Features

  • ✅ Add, edit, and delete tasks easily
  • 📂 Categorize tasks as Study, Assignment, or Personal
  • 📅 Set due dates to track deadlines
  • 🔄 Mark tasks as completed or pending
  • 💾 Data persists in browser using localStorage
  • 📱 Fully responsive design for all devices
  • 🚀 No login required — works instantly
